Headings

in html:
    <h1>Heading 1</h1>
    <h2>Heading 2</h2>
    <h3>Heading 3</h3>
in markdown:
    # Heading 1
    ## Heading 2
    ### Heading 3

Paragraphs

in html:
    <p>This is a paragraph</p>
in markdown:
    This is a paragraph

Bold

in html:
    <p>This is a <b>bold</b> word.</p>
in markdown:
    This is a **bold** word.

Italic

in html:
    <p>This is an <i>italic</i> word.</p>
in markdown:
    This is an *italic* word.

Links

in html:
    This is a paragraph with a <a href="https://www.google.com">link</a>.
in html:
    This is a paragraph with a [link](https://www.google.com).

Rendering images

in html:
    <img src="url/of/image.jpg" alt="Description of image">
in markdown:
    ![alt text for image](url/of/image.jpg)

Orderless lists

in html:
    <ul>
        <li>point 1</li>
        <li>point 2</li>
        <li>point 3</li>
    </ul>
in markdown:
    * point 1
    * point 2
    * point 3

Ordered lists

in html:
    <ol>
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
    </ol>
in markdown:
    1. Item 1
    2. Item 2
    3. Item 3

Quotes

in html:
    <blockquote>
        This is a quote.
    </blockquote>
in markdown:
    > This is a quote.

Code

in html:
    <code>This is code</code>
in markdown:
    ```
    This is code
    ```
